# MacroPad-for-Productivity
A MacroPad that will help enhance my workflow and productivity in writing documents.

This project was my introduction into hardware design with PCB. I would say this was pretty hard but after learning it I was able to become way more knowledgeable. I enjoyed this project a lot!

# Firmware
I used QMK for the macropad's firmware. I made the 5 keys have productivity enhancement commands such as copy and paste. The rotary encoder will be used for volume control.

# BOM (Bill of Materials)
The case parts can be 3D printed and the PCB can be attained from a pcb manufactuer like JLCPCB.
|Quantity|Material|
|---|---|
|1|Seeed XIAO RP2040|
|1|EC11 Rotary Encoder|
|4|M3x16mm Screws|
|5|MX-Style Switches|
|5|White Blank DSA Keycaps|
